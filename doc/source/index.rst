Welcome to Zebra's documentation!
=================================

Zebra is a backup and recovery tool for ZFS pools and datasets. It is designed
to run as command line application and heavily uses ZFS specific features such
as snapshots, or the ability to send and receive a dataset's content.

Zebra is developed and maintained by the `RootForum.org`_ team. It is open
source and subject to the conditions of the MIT license.
